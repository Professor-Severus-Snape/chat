# Чат

Простое браузерное приложение для обмена сообщениями в реальном времени.  
Чат построен на WebSocket и позволяет пользователям общаться без перезагрузки страницы.

![](./pic/chat-2.png)

## Возможности

- Подключение пользователя по уникальному никнейму
- Отправка сообщений в реальном времени
- Отображение списка активных пользователей
- Разделение сообщений на свои и чужие
- Автоматическое обновление интерфейса
- Удаление пользователя из списка при отключении

## Технологии

- JavaScript (ES6+, ООП)
- DOM API
- CSS
- Webpack
- ESLint (Airbnb config)
- Node.js
- Express
- WebSocket
- REST API


## CI/CD

- GitHub Actions - линтинг и сборка клиентской и серверной частей приложения (CI)
- GitHub Pages - деплой клиентской части приложения (CD)
- Render.com - хостинг серверной части приложения (CD)

## Архитектура

Приложение построено по клиент-серверной архитектуре.

Клиент устанавливает WebSocket-соединение с сервером.  
Сервер управляет списком пользователей и пересылает сообщения всем участникам чата.

## Структура проекта

- `client/` — клиентская часть приложения (frontend)
- `server/` — серверная часть приложения (backend)
- `pic/` — скриншоты интерфейса
