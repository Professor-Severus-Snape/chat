# Backend

Серверная часть приложения «Чат».  
Описание проекта доступно в [главном README](../README.md).

## Назначение

Сервер обеспечивает работу чата в реальном времени.

Основные задачи:

- регистрация пользователей
- проверка уникальности никнеймов
- хранение списка пользователей онлайн
- обработка WebSocket-подключений
- рассылка сообщений всем участникам чата
- обновление списка пользователей при подключении и отключении

## Архитектура взаимодействия

Сервер использует два типа взаимодействия:

**REST API**

- регистрация пользователя
- проверка доступности сервера

**WebSocket**

- получение сообщений
- отправка сообщений
- обновление списка пользователей

## API

### REST API

- **GET /ping-server** — проверка доступности сервера
- **POST /new-user** — регистрация нового пользователя (параметр: `name`)

### WebSocket API

- **ws://localhost:7070/ws** — подключение к локальному серверу
- **wss://ahj-websockets-backend.onrender.com/ws** — подключение к серверу на render.com

- **type: send** — отправка сообщения
- **type: exit** — выход пользователя

## Технологии

- Node.js
- Express
- WebSocket
- REST API
