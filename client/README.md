# Frontend

Клиентская часть приложения «Чат».  
Описание проекта доступно в [главном README](../README.md).

## Назначение

Клиент отвечает за пользовательский интерфейс и взаимодействие с сервером через WebSocket.

Основные задачи:

- регистрация пользователя по никнейму
- подключение к WebSocket-серверу
- отправка сообщений
- отображение сообщений в чате
- отображение списка активных пользователей
- обновление интерфейса при подключении или отключении участников

## Интерфейс

После загрузки страницы пользователь вводит никнейм.

![](../pic/chat-1.png)

После успешной регистрации открывается окно чата.

![](../pic/chat-2.png)

Особенности интерфейса:

- собственные сообщения отображаются справа
- сообщения других пользователей — слева
- список активных пользователей отображается в боковой панели

## Технологии

- JavaScript (ES6+)
- DOM API
- REST API
- CSS
- Webpack
- ESLint (Airbnb config)
